Ejercicios de Variables en JavaScript

Ejercicio 1: Datos Personales

Declara tres variables para nombre, edad y ciudad.

Ejercicio 2: Valor de Pi

Almacena el valor de pi (3.1416) en una variable.

Ejercicio 3: Número y String

Muestra un número y un string juntos.

Ejercicio 4: Multiplicación

Multiplica dos números y almacena el resultado.

Ejercicio 5: Pide tu Edad

Ingresa tu edad y muestra un mensaje personalizado.

Ejercicio 6: Nombre Completo

Ingresa tu nombre y apellido.

Nota: Abre la consola del navegador (F12) para ver los resultados en console.log()
