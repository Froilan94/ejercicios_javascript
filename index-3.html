Ejercicios de Funciones en JavaScript

Ejercicio 1: Suma de Dos Números

Función que suma dos números y retorna el resultado.

Ejercicio 2: Saludo Personalizado

Función que recibe un nombre y muestra un saludo.

Ejercicio 3: Área de un Círculo

Función que calcula el área de un círculo (π × r²).

Ejercicio 4: Número Mayor

Función que retorna el mayor de dos números.

Ejercicio 5: Par o Impar

Función que retorna true si es par, false si es impar.

Ejercicio 6: Promedio de un Array

Función que calcula el promedio de números en un array.

(Separados por comas)
